# SmileCare - Dental Clinic Management System

This is a web-based application for managing a dental clinic, including appointments, patient records, and doctor information.

## Features

- User authentication (Admin, Doctor, Patient)
- Appointment scheduling
- Patient management
- Doctor management
- Medical history tracking

## Setup Instructions

1. Clone the repository.
2. Set up a web server (e.g., Apache with XAMPP).
3. Import the `dentalcare.sql` file into your MySQL database.
4. Update database connection details in `dc/include/config.php`.
5. Access the application through your web server.

## Technologies Used

- PHP
- MySQL
- HTML
- CSS (Bootstrap)
- JavaScript (jQuery)
